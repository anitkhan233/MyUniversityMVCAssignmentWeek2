An ASP.NET MVC program for University consists of Students, Teachers, Units, Courses

Load below URL in the browser (Use your own respective port number) to view the outcome:

https://localhost:44387/StudentCourseUnitTeacherViewModels/Index

Note that "MVC 5 Controller with views, using Entity Framework" was used to automatically create controllers and views which also created respective tables in the database based on the respective models.

Data has been stored and retrieved from database "MyUniversityMVCAssignmentWeek2Database".
